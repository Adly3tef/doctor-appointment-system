# 🏥 Doctor Appointment System

A full-stack web application for booking doctor appointments, built with the **MEAN Stack**.

![Angular](https://img.shields.io/badge/Angular-17-red) ![Node.js](https://img.shields.io/badge/Node.js-Express-green) ![MongoDB](https://img.shields.io/badge/Database-MongoDB-brightgreen) ![JWT](https://img.shields.io/badge/Auth-JWT-blue)

## ✨ Features

- 🔐 Secure login & registration with JWT authentication
- 👤 User dashboard with appointment history
- 📅 Book & cancel appointments with doctors
- 🏥 Browse medical departments and specialists
- ⚙️ Admin dashboard to manage doctors & departments
- 🛡️ Role-based access control (User / Admin)
- 📱 Fully responsive design

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Angular 17, TypeScript, CSS3 |
| Backend | Node.js, Express.js |
| Database | MongoDB, Mongoose |
| Authentication | JWT, Bcrypt |
| Architecture | RESTful API |

## 🚀 Getting Started

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd forntend/hospital-frontend
npm install
ng serve
```
